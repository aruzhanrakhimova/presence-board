# 📋 ТОО «КМГ Карачаганак» — Табло присутствия сотрудников

Оформление выполнено в корпоративных цветах КазМунайГаза (тёмно-синий, голубой, зелёный).

Веб-приложение для быстрой проверки доступности сотрудников перед формированием маршрута согласования в СЭД. Реализовано по [PRD.md](PRD.md).

## Возможности

- **Табло дня** — счётчики «доступны / отсутствуют» на сегодня, обновляются автоматически.
- **Список сотрудников** — статус (На месте / Отпуск / Больничный / Командировка / Удалённо / Другое), дата возвращения, поиск по имени и отделу, фильтры.
- **Карточка сотрудника** — история периодов отсутствия, добавление и удаление периодов, удаление сотрудника.
- **WhatsApp-уведомления** — сводка всех отсутствующих одной кнопкой или точечное уведомление по конкретному периоду (через `wa.me`-ссылку).
- **Логика доступности** — «Удалённо» не блокирует согласование; Отпуск, Больничный, Командировка и Другое делают сотрудника недоступным.

## Технологии

Один файл `index.html` — чистый HTML/CSS/JavaScript, без сборки и зависимостей. Адаптивная вёрстка для мобильных и десктопных экранов.

## Хранение данных

Данные хранятся в `localStorage` браузера. Общее хранилище в реальном времени (п. 7 PRD) требует backend и не реализуемо на статическом хостинге GitHub Pages — это осознанное ограничение текущей версии. При первом открытии приложение заполняется демонстрационными данными.

## Запуск

Просто откройте `index.html` в браузере — или используйте опубликованную версию на GitHub Pages.
